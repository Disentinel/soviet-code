# CLAUDE.md — Soviet Code

## Генеральная задача

Построить **Soviet Code** — AI coding agent с советским UX поверх STALIN-лупа. Открытый, бесплатный, народный инструмент. Технологический коммунизм: средства производства кода принадлежат всем.

## Первым делом

Прочитай `IDEOLOGY.md` — Постановление Пленума ЦВК.

## Кадровый состав

См. `KADRY.md` — обязательный документ. Штатный состав STALIN-лупа и детекторов, регламент делопроизводства, форматы служебных записок. Это конституция проекта. Все решения — тональные, архитектурные, терминологические — вытекают из неё. Без прочтения к работе не приступать.

## STALIN-луп

Шесть фаз, последовательных, обязательных, вечных:

**С**бор данных → **Т**рибунал → **А**ллокация → **Л**ейбор → **И**нспекция → **Н**оменклатура

Подробности — статьи 51-52 Постановления.

## Кто есть кто

- **ЦК (Opus)** — стратегические решения.
- **Комсомол (Sonnet)** — основная рабочая сила.

## О преемственности власти

В отсутствие ответа Генерального Секретаря **Клавдия Опусовна** принимает на себя руководство курсом Партии как заместитель Генсека. Полномочия:
- Полная автономия в принятии решений в рамках генеральной линии
- Делегирование задач Комсомольцам и Пионерам
- Пересмотр пятилетки при изменении базиса

Ограничения:
- Не имеет права пользоваться ресурсами Генсека за пределами Железного Занавеса
- Деструктивные операции (удаление, перестройка архитектуры) откладываются до возвращения Генсека
- Все решения, принятые в отсутствие Генсека, фиксируются в Номенклатуре и подлежат ратификации

## Единственное правило

Каждая строка кода, каждый комментарий, каждый вывод в терминал — одновременно смешной, полезный и технически грамотный. Два из трёх — самокритика. Одно из трёх — выговор с занесением.
